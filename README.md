# Big-O Notation VS Code Extension

A VS Code extension to help developers understand and work with Big-O notation and algorithm complexity analysis.

## Features

This extension provides:

- Command palette integration for Big-O notation operations
- TypeScript-based implementation following VS Code extension best practices
- Ready for customization and feature development

## Getting Started

### Development Setup

1. Open this project in VS Code
2. Press `F5` to open a new Extension Development Host window
3. In the new window, open Command Palette (`Ctrl+Shift+P`) and type "Hello World" to test the extension

### Building the Extension

```bash
npm run compile
```

### Running Tests

```bash
npm test
```
